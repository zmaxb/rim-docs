РИМ не содержит готовых ботов для реального использования, однако включает тестовый бот, который можно использовать для проверки работы системы.

### ▶️ Общий сценарий

1. Подключается мессенджер  
2. Настраивается бот  
3. РИМ принимает входящие события  
4. Бот обрабатывает их  
5. При необходимости формируется и отправляется ответ

Бот — это основной способ реализации логики в РИМ.
Он обрабатывает входящие события и определяет, как система должна на них реагировать.
